### [백준 1806번](https://www.acmicpc.net/problem/1806)

**풀이**

```python3
n, s = map(int, input().split())
INF = int(1e9)
res = INF

arr = list(map(int, input().split()))
i = j = 0
total = 0
while i < n:
    cnt = j-i
    while j < n and total < s:
        total += arr[j]
        j += 1
        cnt += 1
    if total >= s:
        res = min(res, cnt)
        total -= arr[i]
        i += 1
    else:
        break

if res == INF:
    print(0)
else:
    print(res)
```
