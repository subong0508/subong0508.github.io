### [백준 1449번](https://www.acmicpc.net/problem/1449)

**풀이**

```python3
n, l = map(int, input().split())
arr = list(map(int, input().split()))
arr.sort()

cnt = 1
start = arr[0]-0.5
for i in range(1, n):
    end = arr[i]+0.5
    # 돌려막기 가능
    if end <= start+l:
        continue
    else: # 새로 테이프를 붙여야 한다.
        cnt += 1
        start = arr[i]-0.5

print(cnt)
```
