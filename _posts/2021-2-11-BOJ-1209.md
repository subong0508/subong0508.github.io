### [백준 2109번](https://www.acmicpc.net/problem/2109)

**풀이**

```python3
n = int(input())
arr = [-1] * n
for i in range(n):
    p, d = map(int, input().split())
    arr[i] = (p, d)
# 강연비순으로 정렬
arr.sort(key=lambda x: x[0])
filled = [False] * 10001

res = 0
while arr:
    p, d = arr.pop()
    # 빈 날 있는지 찾아보기
    while d >= 1 and filled[d]:
        d -= 1
    # 순회강연을 할 수 있는 경우
    if d >= 1:
        filled[d] = True
        res += p

print(res)
```
