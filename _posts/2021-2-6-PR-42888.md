### [프로그래머스 42888번](https://programmers.co.kr/learn/courses/30/lessons/42888)

**풀이**

```python3
def solution(record):
    msgs = []
    user_d = {}
    for r in record:
        lst = r.split()
        if len(lst) == 2:
            continue
        action, uid, name = lst
        user_d[uid] = name
    
    for r in record:
        lst = r.split()
        if len(lst) == 2:
            _, uid = lst
            msg = user_d[uid] + "님이 나갔습니다."
            msgs.append(msg)
        else:
            action, uid, name = lst
            if action == "Enter":
                msg = user_d[uid] + "님이 들어왔습니다."
                msgs.append(msg)
    return msgs
```
